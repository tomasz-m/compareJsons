Compare JSONs

Check if structure of two JSON strings is same. The content is ignored, just fields and its types matter.

For example comparing this...

{
  "Person": [
    {
      "Name": "John",
      "Age": 3
    },
    {
      "Name": "Fred",
      "Age": 45
    }
  ]
}

...to this...

{
  "Person": [
    {
      "Age": "32",
      "Name": "Arnold"
    }
  ]
}

...will show you
